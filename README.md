# pivot
a command line tool to transpose CSV like text

Simple and useful

```
  $ git clone https://github.com/Grijesh-Chauhan/pivot.git /tmp/repo
  $ mkdir ~/bin
  $ mv /tmp/repo/pivot ~/bin/.
  $ chmox a+x
  $ echo 'export PATH="$HOME/bin:$PATH"' >> ~/.bashrc
  $ pivot --examples
```

Give it a Try!!
